# Resources (raw sources)

This folder holds raw source datasets used by LV0 ingest. Files here are not
committed to Git by default.

Raw data flow reference: `docs/RAW_DATA_FLOW.md`.

## Current sources

- english_modern
  - kaikki.org-dictionary-English.jsonl
  - cmudict.dict
- english_middle
  - kaikki.org-dictionary-MiddleEnglish.jsonl
- english_old
  - kaikki.org-dictionary-OldEnglish.jsonl
- latin
  - kaikki.org-dictionary-Latin.jsonl
- ancient_greek
  - kaikki.org-dictionary-AncientGreek.jsonl

## Notes

- Kaikki.org exports are Wiktionary-derived JSONL.
- cmudict.dict provides modern English pronunciations (ARPAbet).
