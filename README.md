# Juthoor Linguistic Genealogy

Unified monorepo for the LV0-LV4 research program (Qur'anic semantics to cross‑linguistic genealogy).

## Project Structure

This monorepo is organized into a layered architecture:

- `Juthoor-DataCore-LV0/` (LV0: Foundation & Data Engine)
- `Juthoor-ArabicGenome-LV1/` (LV1: Arabic Root System Decoding)
- `Juthoor-CognateDiscovery-LV2/` (LV2: Cross-Lingual Discovery Engine)
- `Juthoor-Origins-LV3/` (LV3: Genealogical Theory & Mapping)
- `Quran-Corpus-Analysis/` (Application: Quranic Semantic Analysis)

## Workspace
- Root `pyproject.toml` defines a workspace with `Juthoor-DataCore-LV0` and `Juthoor-CognateDiscovery-LV2` as packages.
