## The Missing Principle: Arabic Has Maximum Phonetic Resolution

اللسان العربي أكثر فصاحة — Arabic is the most phonetically precise Semitic language. It distinguishes sounds that **other Semitic languages merged**:

| Arabic Distinction | Merged In | Example |
|---|---|---|
| ع (ʿayn) vs غ (ghayn) | Hebrew, Aramaic | ע covers both |
| ح (ḥāʾ) vs خ (khāʾ) | Hebrew (partially) | ח covers both |
| ق (qāf) vs ك (kāf) | Many dialects, some Semitic | Often merged |
| ض (ḍād) vs ظ (ẓāʾ) | Most Semitic languages | Unique to Arabic |
| ت (tāʾ) vs ط (ṭāʾ) | Hebrew (partially) | ת covers both |
| د (dāl) vs ذ (dhāl) | Hebrew | ד covers both |
| س (sīn) vs ص (ṣād) | Hebrew (partially) | Merged in some contexts |

**This means:** when comparing Arabic to Hebrew/Aramaic/Persian, you CANNOT assume letter-for-letter correspondence. You must account for **which Arabic distinctions the target language lost**.

## Key Corrections

### ערב/عرب — Hebrew merged ع/غ

- Hebrew ערב (evening) = Arabic **غرب** (gharb = west, المغرب = sunset = evening)
- Hebrew does NOT distinguish ع from غ — both map to ע
- Arabic عرب (Arab) and غرب (west) are **two different roots**
- Hebrew collapsed them into one because it lost the ع/غ distinction

### حمار/חמרא — Different letters ح ≠ خ

- حمار (ḥimār, with **ح**) = donkey → root ح-م-ر
- خمر (khamr, with **خ**) = wine → root خ-م-ر
- These are different Arabic letters with different semantic fields
- Hebrew/Aramaic maps both to ח because they merged ح/خ

### خورشید — There IS a root connection

- Avestan **Hvar** (sun) maps to Arabic **حور** (ḥ-w-r = radiance/visual brilliance)
- The connection exists through Proto-Indo-Iranian ↔ Semitic consonantal parallels

## Rules for Benchmark Construction

### For Gold Pairs:
1. Map consonants correctly — account for mergers in target language
2. Use Arabic as reference — highest phonetic resolution
3. Check ALL possible Arabic source roots for merged consonants

### For Negatives:
1. **Zero consonant overlap** is the only safe criterion
2. Cross-family pairs (Semitic vs unrelated) are safest
3. Within Semitic, be extremely cautious

### For False Friends — Checklist before labeling:
1. Does the target language merge two Arabic sounds? → Check BOTH Arabic sources
2. Does the dotted/undotted distinction explain the "different" meanings?
3. Does Ibn Faris's principle reconcile surface divergence?
4. Does Jabal's binary root theory connect through a shared ثنائية?

Only if ALL four checks fail → label as false friend.
